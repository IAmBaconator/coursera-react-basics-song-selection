# coursera-react-basics-song-selection
A demo application to show how the React Player works by showing two buttons that can toggle play/pause.
## Goal
To show how the React Player works by displaying a list of buttons and their controls.
# Lab Notes
Command to create a _(this)_ React app, `npx create-react-app song_selection` and start the browser preview of the app, `npm start`.<br />
**Note:** The create-react-app command is deprecated.<br />
Command to install the React Player Media Package, `npm install react-player`
